# Keyframes

All the keyframes from the plugin that enable [/classes/animation-utilities](./classes.md#animations-utilities). Although keyframes cant be used directly as classes, you can use them inside your own `.(s)css` files if you opt for manual animation instead of [/classes/animation-utilities](./classes.md#animations-utilities).

- `fadeInAnimation`
- `fadeOutAnimation`
- `scaleInAnimation`
- `scaleOutAnimation`
- `slideUpAnimation`
- `slideDownAnimation`
- `pulseGlowAnimation`
